3. 查询方法一：
           select* from test where instr((select  group_concat(sub_msgs) from test where sub_msgs!=""),id)>0;
          instr ，返回子串substr在字符串str中的第一个出现的位置。
          GROUP_CONCAT(courses_id SEPARATOR '|||')，默认用英文逗号分割，SEPARATOR色值分割符


4. 查询方法二：
        select* from test where FIND_IN_SET(id,(select  group_concat(sub_msgs) from test where sub_msgs!=""));
       FIND_IN_SET，假如字符串str 在由N 子链组成的字符串列表strlist 中，则返回值的范围在 1 到 N 之间。
一个字符串列表就是一个由一些被‘,’符号分开的子链组成的字符串。如果第一个参数是一个常数字符串，而第二个是type SET列，则   FIND_IN_SET() 函数被优化，使用比特计算。
如果str不在strlist 或strlist 为空字符串，则返回值为 0 。如任意一个参数为NULL，则返回值为 NULL。这个函数在第一个参数包含一个逗号(‘,’)时将无法正常运行   

4. 总结：第一种方法可能出现错误数据，因为只是比较后面的数据是否在前面出现过，例如，“1”在“11”中出现过。
