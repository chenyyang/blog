只能用byte做中间转换，如果是string会有丢失：
      
        File file = new File("/homeDownloads/b.zip");
        byte[] lines = FileUtils.readFileToByteArray(file);

        System.out.println(Base64.encode(lines));

        File fileName = new File("/homeDownloads/bcopy.zip");
        PrintWriter outFile = null;
        outFile = new PrintWriter(fileName);

        FileUtils.writeByteArrayToFile(fileName, lines);
        outFile.flush();
        outFile.close();
