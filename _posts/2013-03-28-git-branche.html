同步远程分支（直接）：
        git pull origin [分支名] ：git pull origin master (将master分支的代码获取下来并merge到当前分支上，但不push，如果有冲突会提示冲突)
同步远程分支（先查看后提交）：
        git fetch origin master （从远程分支上下载代码）
        git log -p test..origin/master （本地test分支和远程master分支进行比较，查看之间提交了什么修改）
        git merge origin/master（远程master分支进行merge到本地test分支）
