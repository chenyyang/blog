1. 从单个文件中搜索指定的字串
语法:
grep "string" filename
例子:
grep "name" file.txt
2. 在多个文件中检索指定的字串
语法:
grep "string" FILE_PATTERN
例子:
grep "name" file_*
3. 用 grep -i 进行大小写无关的搜索
语法:
grep -i "string" FILE
例子:
grep -i "name" file.txt
4. 使用用正则表达式
语法:
grep -E "REGEX" filename
例子:
grep -E "name.*empty" file.txt
5. 用grep -w搜索整个词，而不是词中的部
语法:
grep -w "string" FILE
例子:
grep -iw "is" demo_file 搜索包含大写或小写is单词的行
6. 使用grep -A, -B and -C显示之前、之后、前后的几行
语法:
grep -A n "string" FILENAME或者grep "string" FILENAME -A n 匹配行的后面几行
grep -B n "string" FILENAME或者grep "string" FILENAME -B n 匹配行的前面几行
grep -C n "string" FILENAME或者grep "string" FILENAME -C n 匹配行的前面后面几行
7.通过--color 高亮显示搜索的字串
语法:
grep this file.txt --color
例子:
grep "name" file.txt --color
8. 用grep -r递归搜索全部的文件
语法:
grep -r "string" FILE
例子:
grep -r "name" *
12. 用grep -l 只显示文件名
语法:
grep -l "string" FILE
例子:
grep -l name file_*
13. 用 grep -n 在输出时显示行号
语法:
grep -n "string" FILE
例子:
grep -n "name" file.txt
14. 使用grep -v进行不匹配
语法:
grep -v "string" FILE
例子:
grep -v "name" file.txt
